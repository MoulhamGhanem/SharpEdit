# SharpEdit 📝

SharpEdit is a lightweight, intuitive text editor built with C# and WinForms. It provides a clean interface for basic text manipulation and file management.

## ✨ Features
- **File Operations:** Create, Open, Save, and Save As `.txt` files.
- **Unsaved Changes Warning:** Never lose your work! The app detects unsaved changes and prompts you to save before exiting or opening a new file.
- **Visual Indicators:** Real-time asterisk (*) indicator in the title bar for modified files.
- **Rich Text Support:** Uses `RichTextBox` for a smooth writing experience.

## 🚀 How to Run
1. Clone the repository:
   git clone [https://github.com/MoulhamGhanem/SharpEdit.git](https://github.com/MoulhamGhanem/SharpEdit.git)

Open the .sln file in Visual Studio.

Press F5 to build and run the application.
